📋 포스(POS) 매장 장애 대응 가이드

📅 작성일: 2026.03.03 (1.0ver) 👥 대상: 경영지원팀 / 운영팀 / 매장 관리자 🎯 소프트웨어팀 없이 현장 1차 대응 가이드
① 증상 확인 → ② 아이디 확인 → ③ 경영지원 알림 → ④ 1차 대응 → ⑤ 비상 운영

1. 장애 유형별 대응 매뉴얼

1-1. 인터넷 장애 시

🔴 증상
  • 포스 화면에 "네트워크 연결 실패" 또는 "서버 연결 안됨" 표시
  • 결제 처리 안됨 / 메뉴 로딩 안됨
⚠️ 사전 확인
  • 매장 아이디 및 스태프 아이디가 정상인지 확인
  • 위 확인 후에도 해결되지 않으면 → 경영지원/운영팀/직영점 매니저가 함께 1차 대응 진행
🔵 1차 대응 (경영지원 / 운영팀 / 직영점 매니저)
  1. 공유기(라우터) 확인
    • 공유기 전원 OFF → 10초 대기 → 전원 ON
    • 공유기 표시등 확인 (정상: 녹색 점등 / 이상: 빨간색 또는 소등)
  2. 랜선 확인
    • 포스 ↔ 공유기 랜선 빠져있는지 확인
    • 랜선 뽑았다가 딸깍 소리 날 때까지 다시 꽂기
  3. 다른 기기 인터넷 확인
    • 매장 내 다른 기기(휴대폰 와이파이 등)로 인터넷 되는지 확인
    • 다른 기기도 안 되면 → 통신사 문제 (통신사 고객센터 연락)
    • 다른 기기는 되는데 포스만 안 되면 → 포스 네트워크 설정 문제
🟢 비상 운영 (플랜B)
  • 수기 주문서로 전환 (별도 양식 비치)
  • 주문 내용, 시간, 금액 기록
  • 현금/계좌이체 결제 유도 (카드결제 불가 시)
  • 포스 복구 후 → 수기 내용을 포스에 일괄 입력

1-2. 포스 프로그램 오류 시

🔴 증상
  • 포스 프로그램 먹통 / 화면 멈춤
  • 버튼 클릭 반응 없음
  • 오류 메시지 팝업
⚠️ 사전 확인
  • 매장 아이디 및 스태프 아이디가 정상인지 확인
  • 위 확인 후에도 해결되지 않으면 → 경영지원/운영팀/직영점 매니저가 함께 1차 대응 진행
🔵 1차 대응 (경영지원 / 운영팀 / 직영점 매니저)
  1. 프로그램 강제 종료 후 재시작
    • Ctrl + Alt + Delete → 작업관리자 → 포스 프로그램 선택 → 작업 끝내기
    • 바탕화면에서 포스 프로그램 다시 실행
  2. 컴퓨터 재부팅
    • 프로그램 재시작으로 안 되면 컴퓨터 전체 재부팅
    • 시작 → 전원 → 다시 시작
  3. 캐시/임시파일 삭제 (고급)
    • [구체적인 경로 및 방법 추가 필요]
🟢 비상 운영 (플랜B)
  • 수기 주문서 전환 (인터넷 장애 시와 동일)

1-3. 하드웨어(컴퓨터) 이상 시

🔴 증상
  • 컴퓨터 전원 안 켜짐
  • 모니터 화면 안 나옴
  • 이상한 소리 (삐 소리, 팬 소음 과다)
  • 블루스크린 (파란 화면에 오류 코드)
⚠️ 사전 확인
  • 매장 아이디 및 스태프 아이디가 정상인지 확인
  • 위 확인 후에도 해결되지 않으면 → 경영지원/운영팀/직영점 매니저가 함께 1차 대응 진행
🔵 1차 대응 (경영지원 / 운영팀 / 직영점 매니저)
  1. 전원/케이블 확인
    • 전원 케이블 빠져있는지 확인
    • 멀티탭 스위치 ON 확인
    • 모니터 케이블(HDMI/DP) 확인
  2. 전원 초기화
    • 전원 케이블 분리 → 30초 대기 → 다시 연결 → 전원 ON
  3. 블루스크린 발생 시
    • 오류 코드 사진 촬영
    • 전원 버튼 길게 눌러 강제 종료 → 재시작
  4. 모니터만 안 나올 때
    • 모니터 전원 확인
    • 케이블 뽑았다 다시 꽂기
    • 다른 포트에 연결 시도
🟢 비상 운영 (플랜B)
  • 수기 주문서 전환
  • 여유 노트북/태블릿이 있으면 임시 포스로 활용 검토

1-4. 단말기/플레이엠디 충돌 시

🔴 증상
  • 단말기와 플레이엠디 프로그램 충돌
  • 마감 처리 불가
  • 프로그램 간 간섭으로 정상 동작 안됨
⚠️ 사전 확인
  • 매장 아이디 및 스태프 아이디가 정상인지 확인
  • 위 확인 후에도 해결되지 않으면 → 경영지원/운영팀/직영점 매니저가 함께 1차 대응 진행
🔵 1차 대응 (경영지원 / 운영팀 / 직영점 매니저)
  1. 충돌 프로그램 강제 종료
    • Ctrl + Alt + Delete → 작업관리자
    • 플레이엠디 관련 프로세스 모두 종료
    • 포스 프로그램도 종료
    • 포스 프로그램만 먼저 재시작
  2. 드라이버 삭제 및 재설치 (CH341SER.EXE)
    • 원인: USB 단말기 드라이버가 중복 설치되어 충돌 발생
    • USB 포트에서 단말기 분리
    • CH341SER.EXE 실행 → UNINSTALL 클릭 (기존 드라이버 제거)
    • 동일 파일에서 INSTALL 클릭 (드라이버 재설치)
    • 컴퓨터 재부팅 후 단말기 재연결
  3. 기본 재설치
    • 플레이엠디 프로그램 제거 (제어판 → 프로그램 제거)
    • 재설치 파일로 새로 설치
    • [재설치 파일 위치/다운로드 경로 추가 필요]
🟣 예방 조치
  • 포스와 플레이엠디 동시 실행 시 실행 순서 통일
  • [권장 실행 순서 추가 필요]

2. 비상 운영 체계 (플랜B 종합)

수기 운영 전환 기준

  • 포스 장애 발생 후 10분 이내 복구 불가 시 수기 전환

수기 운영 절차

  1. 비치된 수기 주문서 사용
  2. 기록 항목: 주문시간 / 메뉴명 / 수량 / 금액 / 결제수단
  3. 현금 또는 계좌이체 결제 유도
  4. 카드결제 필요 시 → 별도 카드단말기 사용 (있는 경우)

복구 후 처리

  1. 포스 시스템 정상화 확인
  2. 수기 기록 → 포스에 일괄 입력
  3. 매출 대조 확인
  4. 장애 보고서 작성 (시간/원인/대응/결과)
